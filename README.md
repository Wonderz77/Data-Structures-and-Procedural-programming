Array Algorithms
Overview

This project contains two simple algorithms using arrays:

Sum of Distinct Elements – Finds the sum of elements that are unique to each of two arrays.

Dot Product & Orthogonal Vectors – Calculates the dot product of two vectors and checks if they are orthogonal.

Usage

Problem 1: Sum of Distinct Elements

const set1 = [3, 1, 7, 9];
const set2 = [2, 4, 1, 9, 3];

sumDistinct(set1, set2); // 13


Problem 2: Dot Product & Orthogonal Vectors

const vectorPairs = [
  [[1, 0], [0, 1]],
  [[1, 2], [2, 1]]
];

checkOrthogonal(vectorPairs); // [true, false]
